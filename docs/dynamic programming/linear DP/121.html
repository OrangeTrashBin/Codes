121. Best Time to Buy and Sell Stock

Difficulty: Easy

Link to Leetcode​

Here's the link to the Leetcode Page

Relationships and Memories​

Assume we define that dp[x]dp[x]dp[x] means the highest profit we could get if sell the stock no later than the xxxth day. Then the value of dp[x]dp[x]dp[x] is either the stock price on the xxxth day minus the lower stock price or dp[x−1]dp[x-1]dp[x−1]

Namely, dp[x]=max⁡(dp[x−1],prices[x]−minprice)dp[x] = \max(dp[x-1],prices[x]-minprice)dp[x]=max(dp[x−1],prices[x]−minprice)

Solution​

class Solution:
    def maxProfit(self, prices: List[int]) -> int:
        n = len(prices)
        dp = [0]*n
        price_min = prices[0]
        for i in range(1,n):
            dp[i] = max(dp[i-1],prices[i]-price_min)
            price_min = min(prices[i],price_min)
        return dp[n-1]
